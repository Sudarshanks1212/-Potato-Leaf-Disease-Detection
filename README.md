 # 🥔 Potato Leaf Disease Detection 

## 📌 Project Overview

This project is a deep learning-based web application that detects potato leaf diseases from images. It uses a trained CNN model to classify leaf images into different disease categories and helps in early identification for better crop management.

## 🚀 Features
📷 Upload potato leaf images
🤖 Automatic disease classification using CNN
⚡ Real-time prediction
🌐 Interactive web app built with Streamlit
🎯 High accuracy model for classification

## 🧠 Technologies Used
1. Python
2. TensorFlow / Keras
3. NumPy
4. PIL
5. Streamlit

## 📊 Model Information
* Model Type: Convolutional Neural Network (CNN)
* Input: Leaf image
* Output: Disease classification
* Classes: Early Blight, Late Blight, Healthy
